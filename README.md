# Fortnite Hardware Spoofer

Permanent hardware spoofer for Fortnite to bypass EasyAntiCheat (EAC) hardware bans.

## Installation

### Schritt 1: Python installieren
1. Gehe zu https://www.python.org/downloads/
2. Lade die neueste Python-Version herunter (Python 3.11 oder höher)
3. **WICHTIG**: Beim Installieren die Option "Add Python to PATH" aktivieren!
4. Installation abschließen

### Schritt 2: Abhängigkeiten installieren
Öffne eine PowerShell oder CMD als Administrator und führe aus:
```bash
pip install pywin32
```

Oder falls `pip` nicht funktioniert:
```bash
python -m pip install pywin32
```

### Schritt 3: Programm ausführen
- **GUI-Version**: `python spoofer_gui.py` (als Administrator ausführen!)
- **Konsolen-Version**: `python fortnite_spoofer.py` (als Administrator ausführen!)

## What EasyAntiCheat Checks:
- MAC Address (Network Adapter)
- Hard Drive Serial Numbers
- Motherboard Serial Number
- Windows Installation ID
- Registry Keys
- CPU ID
- GPU Serial Number

## Features:
- Permanent spoofing (survives reboots)
- One-click spoof button
- Automatic Windows reinstall preparation
- Fortnite-specific optimization
